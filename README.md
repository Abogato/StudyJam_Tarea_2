# Solucion tarea 2 - Study Jam


Dentro de este repositorio se encuentra la tarea 2 del Study Jam, en el cual
hacemos una calculadora para una aplicacion movil.

## Requerimientos

  * Android Studio 2.3
  * Gradle 2.3
  * minSdkVersion 15

## Uso

Para probar este ejemplo clona este repositorio de la siguiente forma:
>
>     $ git clone https://github.com/Abogato/StudyJam_Tarea_2.git

Dentro de Android Studio:

* File --> New --> Import Project
* Seleccionas la ruta donde hiciste el `clone` del proyecto.
* Build --> Rebuild Project
* Run
